The game is an entertainment game based on logic and luck and is text-based. The user is expected to open safe cells without hitting a land mine. For this, the size of the table is first requested. A map of the desired size is created and mines are assigned at a rate of 1 in 4 of the number of cells. At each step, the following happens: - The user is asked for target coordinates. - It is checked if the target coordinates are in the table. - Then it is checked whether it coincides with a mine. + If it hits a mine, the game is lost. + If safe cells are opened all the way without hitting a landmine, the game is won. - The java programming language is used in the game. - The game is run by calling various methods from the class named minesweeper. - The project is a product of thinking and working on concepts such as variables, arrays, classes, methods, for-while loops, if-else structures, data types and data structures. Oyun mantık ve şansa dayalı bir eğlence oyunudur ve metin tabanlıdır. Kullanıcının mayına denk gelmeden güvenli hücreleri açması beklenir. Bunun için önce tablonun boyutu istenir. İstenen boyutta harita yaratılır ve hücre sayısının 4'te 1'i oranında mayın ataması yapılır. Her adımda şunlar olur: - Kullanıcıdan hedef koordinatlar istenir. - Hedef koordinatın tablo içinde olup olmadığı kontrol edilir. - Ardından mayına denk gelip gelmediği kontrol edilir. + Eğer mayına denk gelirse oyun kaybedilir. + Mayına denk gelmeden safe hücreler sonuna dek açılırsa oyun kazanılır. - Oyunda java programlama dili kullanılmıştır. - Oyun minesweeper isimli class üzerinden çeşitli methodlar çağrılarak çalıştırılır. - Proje; değişkenler, diziler, sınıflar, methotlar, for-while döngüleri, if-else yapıları, veri tipleri ve veri yapıları gibi kavramlar üzerine düşünmenin ve çalışmanın bir ürünüdür.
