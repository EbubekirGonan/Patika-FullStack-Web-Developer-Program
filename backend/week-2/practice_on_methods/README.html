A program has developed that reduces the number received from the user by 5 until it is 0 or negative, and then increases it by 5 until it becomes the first one. Kullanıcıdan alınan sayıyı 0 ya da negatif olana kadar 5'er azaltıp, sonra tekrar ilk haline gelen gelene 5'er artıran program geliştirildi.
